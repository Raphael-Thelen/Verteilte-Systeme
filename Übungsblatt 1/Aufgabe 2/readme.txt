Das gleiche Programm dist-ring.py wird auf allen Geräten ausgeführt.
Verwendung:
  python dist-ring.py n m

Dabei sind:
  n: Der Index der IP-Adresse dieses Geräts in der Liste m (beginnend bei 0)
  m: Die Liste der IP-Adressen aller Geräte (z. B. 192.168.178.11 192.168.178.12)

Wichtig: Das Gerät mit Index 0 erzeugt das erste Token. Es muss als letztes gestartet werden.
